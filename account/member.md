# 会员说明

RugAdapt 提供会员体系，会员可解锁「场景生成」功能及工具箱中的会员专属工具。

## 查看会员信息

登录后，点击顶部信息栏中的**会员等级**（青色下划线文字），弹出会员信息对话框。

### 已是会员

会员信息页显示以下内容：

![会员信息页截图](/rug-adapt-desc/images/member-info.svg)

> ↑ 会员信息页（请替换为实际截图）

| 字段     | 说明                                   |
| -------- | -------------------------------------- |
| 账号     | 当前登录的手机号                       |
| 会员级别 | 当前会员套餐名称                       |
| 注册时间 | 账号创建日期                           |
| 最后登录 | 上次登录时间                           |
| 会员到期 | 会员到期日期（7 天内到期显示红色警告） |

::: warning 即将到期提醒
会员到期时间距今不足 **7 天**时，到期日期会以红色显示，请及时续费。
:::

## 开通会员

### 方式一：通过会员等级入口

点击顶部信息栏的会员等级文字 → 弹出「开通会员」对话框。

![开通会员截图](/rug-adapt-desc/images/member-buy.svg)

> ↑ 开通会员对话框（请替换为实际截图）

**购买步骤：**

1. 在顶部标签栏选择会员套餐（不同套餐对应不同时长/权益）。
2. 页面显示对应价格。
3. 点击「**点击购买**」，生成支付二维码。
4. 使用微信或支付宝扫码完成支付。
5. 支付成功后，对话框自动关闭，会员状态立即生效。

### 方式二：通过「场景生成」页面

进入「场景生成」功能页，非会员状态下会直接显示购买界面，操作步骤相同。

---

## 续费会员

已是会员时，在会员信息页点击「**续费会员**」按钮，会进入购买流程，操作与开通会员相同。
