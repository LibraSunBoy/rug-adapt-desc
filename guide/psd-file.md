# PSD 素材

「PSD 素材」页面提供付费 PSD 模板文件的购买与下载，购买后可获取百度网盘下载链接。

## 页面总览

![PSD素材总览截图](/rug-adapt-desc/images/psd-overview.svg)

> ↑ PSD 素材页面总览（请替换为实际截图）

每个 PSD 素材卡片上会显示：

- 左上角：「点击图片获取」提示
- 右下角：价格标签（已购买 / 正常价格 / 限时折扣价）

---

## 价格说明

| 标签        | 含义                               |
| ----------- | ---------------------------------- |
| 已购买      | 该素材已购买，点击直接获取下载链接 |
| `N` 币      | 正常售价 N 个 Rug 币               |
| 限时 `N` 币 | 限时折扣价，优先显示折扣价         |

---

## 购买 PSD 素材

点击**未购买**的素材，会弹出购买确认对话框。

![PSD购买弹窗截图](/rug-adapt-desc/images/psd-buy.svg)

> ↑ PSD 购买确认弹窗（请替换为实际截图）

弹窗中显示：

- 素材预览图
- 当前账户余额（Rug 币）
- 素材价格（限时特惠价或正常价）

**操作步骤：**

1. 确认账户余额是否充足。
2. 点击「**确认购买**」完成购买。
3. 余额不足时，点击「**充值土拨鼠货币**」先进行充值。

::: tip Rug 币充值
如需充值 Rug 币，请参考 [充值教程](../account/recharge.md)。
:::

---

## 获取下载链接

购买成功后（或已购买的素材），点击素材会弹出百度网盘链接对话框。

![百度网盘链接弹窗截图](/rug-adapt-desc/images/psd-link.svg)

> ↑ 百度网盘链接弹窗（请替换为实际截图）

弹窗中包含：

- **百度网盘链接**：点击输入框或「复制链接」按钮自动复制
- **提取密码**（如有）：点击输入框或「复制密码」按钮自动复制

**操作步骤：**

1. 点击「**复制链接**」复制百度网盘地址。
2. 打开浏览器，粘贴链接并访问。
3. 如有提取密码，点击「**复制密码**」后在百度网盘页面输入。

::: warning 注意
百度网盘链接需在浏览器中打开，不能在插件内直接下载。
:::
