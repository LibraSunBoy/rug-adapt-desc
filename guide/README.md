# 快速开始

欢迎使用 **RugAdapt** —— 一款运行在 Adobe Photoshop 中的 CEP 插件，帮助设计师快速获取素材、管理本地文件，并提供一系列提升效率的 PS 工具。

## 环境要求

| 项目            | 要求                            |
| --------------- | ------------------------------- |
| Adobe Photoshop | CC 2019 及以上版本              |
| 操作系统        | Windows 10 / macOS 10.14 及以上 |
| 网络            | 需要联网（素材下载、账户登录）  |

## 安装插件

1. 下载最新版本的 RugAdapt 安装包（`.zxp` 文件）。
2. 使用 [ZXP Installer](https://aescripts.com/learn/zxp-installer/) 或 [Anastasiy's Extension Manager](https://install.anastasiy.com/) 安装。
3. 重启 Photoshop。
4. 在菜单栏选择 **窗口 → 扩展功能 → RugAdapt** 打开插件面板。

::: tip 提示
安装完成后，如果在扩展功能菜单中找不到插件，请确认 PS 版本是否支持 CEP 扩展，并检查安装是否成功。
:::

## 界面总览

插件面板从上到下依次为：

- **顶部信息栏**：显示账号、会员等级、Rug 币余额、版本号，以及充值、设置、退出按钮。
- **顶部横幅**：可点击跳转到版本更新页面。
- **导航栏**：切换各功能页面（添加素材、本地素材、PSD 素材等）。
- **内容区**：当前功能页面的主体内容。
