# 系统设置

系统设置用于配置插件的本地参数，目前支持设置本地素材文件夹路径。

## 打开设置

点击顶部信息栏右侧的「**设置**」按钮，打开全屏系统设置对话框。

![系统设置截图](/images/setting-dialog.png)

> ↑ 系统设置对话框

---

## 本地素材文件夹路径

在输入框中填写本地样机的根目录路径，用于「本地样机」功能读取本地文件。

**路径示例：**

- Windows：`D:\我的样机` 或 `C:\Users\yourname\Documents\Mockups`
- macOS：`/Users/yourname/Mockups`

填写完成后，点击右下角「**确 定**」保存设置。

::: tip
保存后，「本地样机」页面会自动刷新，读取新路径下的文件夹和文件。
:::
