## 安装步骤

**1. 运行安装程序**

双击下载的 `.exe` 文件，启动安装向导。

**2. 按向导完成安装**

按照提示点击「下一步」完成安装，安装路径保持默认即可。

**3. 重启 Photoshop**

安装完成后，完全退出并重新启动 Adobe Photoshop。

**4. 打开插件**

在 Photoshop 菜单栏选择：**窗口 → 扩展功能 → RugAdapt**
